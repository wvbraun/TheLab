Tuples are an immutable list. 

>>> tuple = ("a", "b", "wut", 2)
>>> tuple[0]
"a"
>>> tuple[-1]
2


You cannot use append(), extend(), remove(), pop()

-----------------
Assigning multiple values at once:

>>> v = ('a',2, True)
>>> (x,y,z) = v 
>>> x 
'a'
>>> y 
2
>>> z 
True

ex:
>>> (mon, tue, wed, thu, fri, sat, sun) = range(7)
>>> mon
0
>>> wed
2
