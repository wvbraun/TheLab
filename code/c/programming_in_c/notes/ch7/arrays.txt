Multidimensional Arrays:

C allows for the use of mulidimensional arrays. 

ex: M[i][j] 
	- the first index number [i] refers to the row number
	- the second index number [j] refers to column

---
ex: the statement sum = M[0][2] + M[2][4]; 
	- adds the value in row 0, col 2 +  the value in 
	row 2, col 4. 

--- 
ex: int M[4][5] = { // array M with 4 rows and 5 columns. 
	
	{10, 5, -3, 17, 82}, 
	{9, 0, 0, 8, -7}, 
	{32, 20, 1, 0, 14},
	{0, 0, 8, 7, 6}
};
